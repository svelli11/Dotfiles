What's new in Enhancer for YouTube™ v3.0.7

Bugs

  • Fixed “Default volume level” applied even when disabled if “Control the volume level with the mouse wheel...” is enabled
  • Fixed playback speed not set to normal speed (x1) for embedded live videos if the default playback speed is different
  • Fixed dark theme and other styling issues caused by recent changes on YouTube
  • Fixed css link in the “debug info” file

New features

  • Videos per row: choose how many videos must be displayed per row (Home, Subscriptions, and some other pages)
  • Videos per row (channel): choose how many videos must be displayed per row on channel pages (Channel > Videos tab)
  • Shorts per row: choose how many shorts must be displayed per row (Home and Subscriptions pages)
  • Shorts per row (channel): choose how many shorts must be displayed per row on channel pages (Channel > Shorts tab)
  • Posts per row: choose how many posts must be displayed per row (Explore > News > Top stories > Latest news posts)

Open the options page and check the “Appearance” section to set your preferences
